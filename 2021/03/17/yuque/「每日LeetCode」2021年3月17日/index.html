Lt228. 汇总区间

228. 汇总区间

给定一个无重复元素的有序整数数组 nums 。
返回 恰好覆盖数组中所有数字 的 最小有序 区间范围列表。也就是说，nums 的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个范围但不属于 nums 的数字 x 。
列表中的每个区间范围 [a,b] 应该按如下格式输出：

  • "a->b" ，如果 a != b
  • "a" ，如果 a == b

提示：

  • 0 <= nums.length <= 20
  • -2 <= nums[i] <= 2 - 1
  • nums 中的所有值都 互不相同
  • nums 按升序排列

思路

用一个 prev 记录区间开始，遍历数组如果下一个数不等于当前数+1，那么就在结果数组中插入数组，判断区间范围是不是 1，如果大于等于 2，按照题意修改为对应格式。
