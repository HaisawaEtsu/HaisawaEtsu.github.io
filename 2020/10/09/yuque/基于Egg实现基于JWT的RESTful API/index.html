前言

本文将基于 Egg.js，使用 Typescript，实现基于 JWT 的 RESTful API。使用到的插件有：egg-mysql、egg-sequelize、egg-validate-joi、egg-jwt、egg-cors、egg-router-plus、egg-swagger-doc
项目源码请访问：https://github.com/HaisawaEtsu/egg-test-demo
